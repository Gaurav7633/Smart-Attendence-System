# Smart-Attendence-System
The Smart Attendance System is a web-based software that will automate and make attendance tracking simple. The interesting educational software employs Face Recognition or QR Code Scanning for attendance in place of a manual system.

The whole project offers a complete solution for institutions and workplaces with a real-time attendance system, secure admin dashboard, and attendance reports. Its goal is to lessen human cooperations, decrease margin of errors and maintain a proper digital record.
